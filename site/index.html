Platos Típicos del Perú API

Consulta platos por región con filtros. Rápida, gratuita y en español.

Comenzar

Uso básico

La API expone rutas públicas:

  • /api lista las regiones
  • /api/<region> platos de la región
  • /api/<region>/<id> detalle de un plato

Ejemplos:

GET /api
GET /api/lima?limit=5
GET /api/cusco?tipo=Fondo&ingrediente=ají
GET /api/la-libertad/12

Parámetros

  • limit: número de resultados (por defecto 20, máx 100)
  • offset: paginado
  • tipo: categoría del plato
  • ingrediente: texto presente en ingredientes
  • q: búsqueda libre por nombre y preparación
  • sort: id, nombre o tipo

Regiones

Recomendaciones

  • Usa los slugs ASCII de región: p. ej., apurimac, la-libertad, madre-de-dios.
  • Aplica filtros moderados para mejorar rendimiento y evitar límites de tasa.
  • Cachea respuestas en tu cliente cuando sea posible.

Ejemplos

JavaScript:

fetch('https://<tu-worker>/api/lima?limit=5')
  .then(r => r.json())
  .then(d => console.log(d))

curl:

curl 'https://<tu-worker>/api/cusco?tipo=Fondo&ingrediente=ají'

Python:

import requests
r = requests.get('https://<tu-worker>/api/arequipa?q=rocoto&sort=nombre&limit=10')
print(r.json())
